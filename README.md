Questa app agenda è stata creata come progetto per il corso di Programmazione ad Oggetti del CdL in Informatica all'Università di Padova nell'a.a. 2025/26. 
L'app permette di creare, modificare, eliminare, salvare e caricare le proprie attività ed è sviluppata interamente in C++, con l'utilizzo del framework Qt, sia per l'implementazione delle classi che per la creazione della GUI.
